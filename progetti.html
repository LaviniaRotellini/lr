MSc Thesis - Digital Humanities

My thesis introduced BiFlow: A Bibliometric Framework for Monitoring Institutional Research Flows. The idea behind the work was using bibliometric data, such as publication dates, affiliation histories, fields of study, to predict how researchers move between institutions. Applied to the Italian academic landscape, BiFlow combined machine learning, explainable AI (XAI), big data analytics and computational linguistics to answer three main questions: - Can researcher mobility be predicted from bibliometric data alone? - Which features carry the most predictive power? - Can explainability improve classification by narrowing the feature space to what actually matters?

Analysis of the Internet Movie Database (IMDb)

This work presents a comprehensive data mining analysis of the IMDb dataset, covering nearly 150,000 titles across multiple analytical tasks. After an extensive preprocessing phase — including missing value imputation via trained classifiers and regressors, outlier detection using LOF, ABOD, and Isolation Forest, and handling of class imbalance through SMOTE, ADASYN, Tomek Links, and ENN — two main classification targets were addressed: titleType and rating. For titleType, models achieved strong results (up to 93% accuracy with XGBoost), while predicting rating proved far harder due to its subjective nature and imbalanced distribution, with best models reaching only around 45% accuracy. SHAP explainability analysis revealed that runtime and number of distribution regions were the most informative features for titleType classification. The paper also includes a regression task on averageRating, a time series analysis of daily box office revenue using motifs, clustering, and shapelet-based classification, and a sequential pattern mining phase, which confirmed that films consistently maintain high revenues in the days immediately following release.

Echo chambers in the subreddit r/DebateVaccines

This work investigates whether echo chambers exist in the Reddit community r/DebateVaccines by analyzing user interactions through social network analysis and natural language processing. A network of over 20,000 users connected by comment interactions was built, then four community detection algorithms were applied (Leiden, Label Propagation, Walktrap, and Demon) to identify clusters. My collegues and I also modeled higher-order interactions using hypergraphs to study how users engage with specific posts. Sentiment analysis classified users as positive, negative, or neutral toward vaccines, revealing a low assortativity score (0.013), meaning users do not strongly prefer interacting with like-minded people. The study concludes that echo chambers do exist in the subreddit, but only in small, tight-knit communities — largely because Reddit's popularity-based recommendation system, unlike Twitter or Instagram, exposes users to a broader range of opinions.

Predictive Analysis on the Open University Learning Analytics Dataset

This work presents a predictive analysis of student outcomes using the Open University Learning Analytics Dataset (OULAD), framed within the broader context o…This paper presents a predictive analysis of student outcomes using the Open University Learning Analytics Dataset (OULAD), framed within the broader context of Technology-Enhanced Learning and Educational Data Mining. I merged seven dataset tables into a unified dataframe, performing extensive preprocessing including handling missing values, discretizing high-granularity variables, and engineering engagement features such as total clicks. A multiclass classification task is then carried out to predict whether a student will pass, fail, or withdraw from a course, using seven machine learning models, of which Gradient Boosting and Random Forest performed best, reaching around 78% and 76% accuracy respectively. A clustering phase using KMeans further segmented students into meaningful groups, later used as a feature in classification. The explainability analysis revealed that the most predictive variables were assessment weight, score, and low engagement with the VLE platform, suggesting that early interventions targeting at-risk students — particularly those with low interaction and prior failed attempts — could meaningfully improve educational outcomes.
